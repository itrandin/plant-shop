# Зеленый уголок

Веб-приложение интернет-магазина комнатных растений.

## Технологии

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- Trefle API
- XAMPP
- phpMyAdmin

## Запуск проекта

1. Установить XAMPP.
2. Запустить Apache и MySQL.
3. Скопировать папку проекта в `htdocs`.
4. Создать базу данных `plant_shop` в phpMyAdmin.
5. Импортировать файл `database.sql`.
6. Импортировать файл `database_views.sql`.
7. При необходимости выполнить `make_admin.sql`.
8. Открыть сайт:

http://localhost/plant-shop

## Функционал

- регистрация и авторизация;
- каталог растений;
- корзина;
- оформление заказа;
- личный кабинет;
- избранное;
- отзывы;
- форма обратной связи;
- административная панель;
- просмотр таблиц и представлений;
- подключение API растений.
